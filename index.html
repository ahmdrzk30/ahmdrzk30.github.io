1. Daily Customers for December 2019

Description : This is a case study from DQLab about a company that wanted to know how the sales projected for December 2019. The data was processed using Python and using Matplotlib to show the result on Visual Studio Code. The line plot was chosen, because it can show the day-to-day sales better.

From the line plot, we know that the highest number of customers was on December 1st. That can be caused by many people getting prepared for Christmas. While the lowest number of customers was around Christmas, and people choose to stay at home with family and friends.

The possible next step that can be taken is analyzing what kind of products sold around the hightest volume of customer, so the company the bring more of the same products around the said time. Another possible step is to analyzing how many customers in certain provinces.


After processing the data by using Python and plot them into bar chart, we can see that DKI Jakarta has the highest number of customers that bought products from the company. Which means, as the nation's capital, DKI Jakarta is indeed high in products consumption's rate.
